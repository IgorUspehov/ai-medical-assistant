# AI Medical Assistant

On-premise AI/RAG architecture prototype for GDPR-safe medical document analysis.

## Features

- PDF processing
- OCR pipeline
- Audio transcription
- RAG search
- Local LLM
- GDPR compliant
- On-premise deployment

## Stack

- FastAPI
- PostgreSQL
- Qdrant
- MinIO
- Docker Compose
- Kubernetes
- vLLM

## Status

Architecture prototype and deployment scaffold.
